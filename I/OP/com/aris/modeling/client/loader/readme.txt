This directory contains the source of the mystic ArisLoader! Beware! Careless changes may result in an unstartable client!

Before submitting any changes, use build_aris_loader.jar.bat in DevTools to create a new aris_loader.jar.

Use this jar file to validate your changes against all of the following use cases:

- Installed client (c:\program files\aris7.1\JavaClient\lib)
- Download client (c:\program files\aris7.1\server\html\lib), test index_app.html AND index_lan.html
- 'Developer client' (get_this_build) (u:\aris\<codeline>\JavaClient\build\dist)
- ESR (details will follow)
